Introduction

CUFSpedia is a CUFS-dedicated open-source site, which spreads information about CUFS so you can learn something about it. CUFSpedia was created on March 25, 2025. I don't know for how long it will last, but I guess it will be here until TCU is alive, or until I have to pay for GitHub, or switch to a paid version of domains.

The site is completely open source; you can press Shift+U to check any data you want. Use the "Contact me" link (just basically a button) to contact me, or join the CUFS Discord server!

History

This idea was honestly born out of boredom. I made this basically just for fun, but hey, it exists now if you see it. My main idea was that I think there will be a lot of people joining, and there will be little information about the group, so I made this. Again, the start date for this site is March 25th, 2025.

Features

Key features of CUFSpedia include its vast collection of articles. I've made this as simple as possible so you can understand and get the idea of CUFS. The main feature is that you can make notes or learn new info about CUFS.

The page also supports various tools for contributors. If you would like to help me in some way, please contact me on Discord.
